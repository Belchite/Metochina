## Local Analysis Only

Metochina is designed to operate entirely on the user's local machine:

- **No data exfiltration**: The tool does not transmit any analyzed data, metadata, or results to external servers, APIs, or third-party services.
- **No network communication**: All analysis is performed locally. The only network-related functionality is the generation of map URL links (e.g., Google Maps, OpenStreetMap) from GPS coordinates, which are presented to the user but not automatically accessed.
- **No telemetry**: The tool does not collect usage statistics, crash reports, or any other telemetry data.
- **User-controlled exports**: Results are only saved to disk when the user explicitly requests an export operation. No data is stored or cached without user action.
